## Run Locally

Clone the project

```bash
  git clone https://github.com/shriram083/Fraazo-Clone.git
```

Go to the project directory

```bash
  cd Assignment 9.6
```

Install dependencies of both frontend and backend

```bash
  cd client
  npm install
  npm start
```

```bash
  cd ../server
  npm install
```

Start the localhost server

```bash
  cd ../client
  npm run start
```
